    **useScrollSpy.js** (per PAGE-03, RESEARCH Pattern 3):
    ```js
    import { useInView } from "react-intersection-observer";

    export function useScrollSpy(id, onInView) {
      const { ref } = useInView({
        rootMargin: "-80px 0px -85% 0px",
        threshold: 0,
        onChange: (inView) => {
          if (inView) onInView(id);
        },
      });
      return ref;
    }
    ```
    CRITICAL rootMargin values: `-80px` top (navbar 64px + 16px buffer), `-85%` bottom (only top 15% of viewport triggers). Do NOT use `-50%` from older ARCHITECTURE.md — use `-85%` from STATE.md and UI-SPEC.md.

    **SidebarTOC.jsx** (per PAGE-03, PAGE-04):
    - `export default function SidebarTOC({ articles, activeId })`
    - Import `colorClasses` from `../../utils/colorClasses`
    - Import `cn` from `../../lib/cn`
    - Outer container: `bg-sidebar-bg rounded-2xl p-6 border border-border-subtle`
    - Title: "In this edition" — `text-[13px] font-extrabold uppercase tracking-wider text-text-3 mb-6`
    - Article list: `<nav className="flex flex-col gap-1">`
    - Each entry is a `<button>` (NOT `<a>` — per STATE.md decision, imperative scroll, no hash URLs):
      ```jsx
      <button
        onClick={(e) => {
          e.preventDefault();
          const el = document.getElementById(article.id);
          if (el) el.scrollIntoView({ behavior: "smooth", block: "start" });
        }}
        className={cn(
          "text-left pl-3 py-2 border-l-2 transition-colors duration-200 text-[14px] leading-snug",
          activeId === article.id
            ? "border-accent text-accent font-extrabold"
            : "border-transparent text-text-3 hover:text-text hover:border-border"
        )}
      >
      ```
    - Inside each button: category dot `<span className={cn("inline-block w-1.5 h-1.5 rounded-full mr-2", colorClasses[article.category]?.dot)} />` + article title text
