## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gaps, dot indicators in category pills |
| sm | 8px | Inline padding within pills, compact meta spacing |
| md | 16px | Default element padding, card content internal gaps |
| lg | 24px | Card border-radius, card content padding |
| xl | 32px | Gap between cards in a row, form element padding |
| 2xl | 48px | TOC entry group spacing, sidebar internal padding |
| 3xl | 64px | Section title-to-content gap |
| 4xl | 100px | Section vertical padding (top + bottom) |

Exceptions:
- Navbar height: 64px (h-16) ‚Äî matches existing pattern from `HomePage.jsx` line 38
- TOC scroll-margin-top: 80px ‚Äî compensates for sticky navbar height (RESEARCH.md, PITFALLS.md)
- Card image height ‚Äî home page hero: min 400px mobile / 500px desktop; secondary cards: 260px mobile / 300px desktop; article grid cards: 180px (CONTEXT.md D-04, existing `HomePage.jsx`)
- Touch targets (buttons, TOC links): minimum 44px height

Source: CONTEXT.md D-11 (100px section padding, 64px gaps, 24px card radius) + REQUIREMENTS.md LAYOUT-07.

## Interaction Contracts

### Card Hover Lift
- Transform: `translateY(-6px)` (per CONTEXT.md D-11)
- Shadow: `0 20px 40px rgba(0,0,0,0.10)`
- Title color: transitions to `--color-accent`
- Duration: 300ms ease-out
- Implementation: Framer Motion `whileHover` with spring `{ stiffness: 400, damping: 30 }`

### Navbar Frosted Glass
- Background: `rgba(255,255,255,0.75)` with `backdrop-blur-md`
- Border bottom: `1px solid --color-border-subtle`
- Firefox fallback: `@supports not (backdrop-filter: blur(1px)) { background: rgba(255,255,255,0.95); }`
- Position: `sticky top-0 z-50`

### TOC Scroll Spy
- Library: `react-intersection-observer` (RESEARCH.md)
- `rootMargin`: `"-80px 0px -85% 0px"` (STATE.md critical note)
- Active state: immediate ‚Äî no debounce
- Scroll on click: imperative `element.scrollIntoView({ behavior: "smooth", block: "start" })` ‚Äî NOT hash URL anchors (STATE.md key decision ‚Äî avoids React Router scroll restoration conflicts)
- `scroll-margin-top: 80px` on each `<section id>` in article body

### Scroll-Reveal Animations
- Above-fold (hero): `initial={{ opacity: 0, y: 20 }} animate={{ opacity: 1, y: 0 }} transition={{ duration: 0.5 }}`
- Cards and sections: `initial={{ opacity: 0, y: 16 }} whileInView={{ opacity: 1, y: 0 }} viewport={{ once: true }} transition={{ duration: 0.4, delay: i * 0.08 }}`
- Implementation: Framer Motion (moved from devDependencies to dependencies ‚Äî REQUIREMENTS.md FOUND-04)

### Section Alternation
- Odd sections: `bg-page-bg` (stone-50)
- Even sections: `bg-section-alt` (white)
- Source: CONTEXT.md D-02, REQUIREMENTS.md LAYOUT-06
